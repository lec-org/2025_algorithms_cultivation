7-2 东河哥哥与水滴 题解

题目理解 : 1.主要理解 :题中延伸的定义为：每一滴水延伸一次可以向任意方向去走一格 , 延伸 k 次再加本身. 单个水			滴延伸k次占格子数为 k + 1 , 最大覆盖排布( 多滴水之间不会互相阻碍，可以理想化地各自发展 )n滴			水就是 n \* (k+1) ;

 		   2.本体数据范围会超int , 须使用long long;

 		   3.还要注意实际覆盖格子数不能超过矩阵总格子数 n × m;

AC代码 :

\#include <bits/stdc++.h>

using namespace std;

\#define int long long



signed main() {

    int n, m, k, q;

    cin >> n >> m >> k >> q;

    int total = n \* m;

    int maxCover = q \* (k + 1);

    cout << min(total, maxCover) << endl;

    return 0;

}
