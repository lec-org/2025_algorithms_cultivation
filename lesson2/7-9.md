# 题目详情：

## 题目描述

在大学期间，经常需要租借教室。大到院系举办活动，小到学习小组自习讨论，都需要向学校申请借教室。教室的大小功能不同，借教室人的身份不同，借教室的手续也不一样。

面对海量租借教室的信息，我们自然希望编程解决这个问题。

我们需要处理接下来 n 天的借教室信息，其中第 i 天学校有 ri​ 个教室可供租借。共有 m 份订单，每份订单用三个正整数描述，分别为 dj​,sj​,tj​，表示某租借者需要从第 sj​ 天到第 tj​ 天租借教室（包括第 sj​ 天和第 tj​ 天），每天需要租借 dj​ 个教室。

我们假定，租借者对教室的大小、地点没有要求。即对于每份订单，我们只需要每天提供 dj​ 个教室，而它们具体是哪些教室，每天是否是相同的教室则不用考虑。

借教室的原则是先到先得，也就是说我们要按照订单的先后顺序依次为每份订单分配教室。如果在分配的过程中遇到一份订单无法完全满足，则需要停止教室的分配，通知当前申请人修改订单。这里的无法满足指从第 sj​ 天到第 tj​ 天中有至少一天剩余的教室数量不足 dj​ 个。

现在我们需要知道，是否会有订单无法完全满足。如果有，需要通知哪一个申请人修改订单。

## 输入格式

第一行包含两个正整数 n,m(1≤n,m≤1e6)，表示天数和订单的数量。

第二行包含 n 个正整数，其中第 i 个数为 ri​(0≤ri​≤1e9)，表示第 i 天可用于租借的教室数量。

接下来有 m 行，每行包含三个正整数 dj​,sj​,tj​(0≤dj​≤1e9,1≤sj​≤tj​≤n)，表示租借的数量，租借开始、结束分别在第几天。

每行相邻的两个数之间均用一个空格隔开。天数与订单均用从 1 开始的整数编号。

## 输出格式

如果所有订单均可满足，则输出只有一行，包含一个整数 0。
否则（订单无法完全满足）输出两行，第一行输出一个负整数 −1，第二行输出需要修改订单的申请人编号。

# 分析：

由于n, m的范围都可以达到1e6所以遍历的话可能会超时。同时想到处理订单具有单调性，例如第k个订单能完成那么前k-1个订单就一定能完成，第k个订单不能完成那么后面的订单也一定不会完成。由此得出要使用二分的方法。然后是设计我们的check函数，遍历每个订单的话会超时，所以我们选择差分数组来进行然后通过前缀和快速得到每天的租借量。

# check函数

```cpp
bool check(int k) { 
	long long diff[N] = {0};//差分数组
	for(int i = 1; i <= k; i ++ )
	{ 
		diff[s[i]] += d[i];
		diff[t[i] + 1] -= d[i];
	} 
	long long current = 0;
	for(int i = 1; i <= n; i ++ ) //计算前缀和
	{ 
		current += diff[i];  
		if(current > r[i]) return false; 
	} 
	return true; 
}
```

# AC代码：
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 1e6 + 10;
int n, m;
int r[N];
int d[N], s[N],t[N];
bool check(int k) {
    long long diff[N] = {0};
    for(int i = 1; i <= k; i ++ ) {
        diff[s[i]] += d[i];
        diff[t[i] + 1] -= d[i]; 
    }
    long long current = 0;
    for(int i = 1; i <= n; i ++ ) {
        current += diff[i];
        if(current > r[i]) {
            return false;
        }
    }
    return true;
}
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);//输入输出优化，关闭同步，加速cin和cout
    cin >> n >> m;
    for(int i = 1; i <= n ;i ++ ) {
        cin >> r[i];
    }
    for(int i = 1; i <= m; i ++ ) {
        cin >> d[i] >> s[i] >> t[i];
    }
    //二分查找
    int l = 0, r = m;
    while(l < r) {
        int mid = l + r + 1 >> 1;
        if(check(mid)) {
            l = mid;
        }else r = mid - 1;
    }
    if(l == m) {
        cout << "0" << endl;
    }else cout << "-1" << endl << l + 1 << endl; 
    return 0;
}
```
