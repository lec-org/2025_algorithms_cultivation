### 7-8 刁总的游戏题解

#### 题目描述

小刁正在玩一个游戏

这个游戏由编号为 1,2,…,*N* 的 *N* 个关卡组成。最初，只能玩第 1 关。对于每个可以玩的关卡 *i*（1 ≤ *i* ≤ *N*−1），你可以在关卡 *i* 执行以下两个操作之一：

- 花费 A~i~ 秒来通关关卡 *i* 。这使你能够玩下一个关卡 X~i+1~
- 花费 B~i~ 秒来通关关卡 *i* 。这使你能够玩关卡 X~C~

忽略除了完成关卡所需时间之外的其他时间，最少需要多少秒才能玩到第 *N* 关？

#### 输入格式

第一行一个整数 *N*
接下来 *N*−1 行每行三个整数 *A*,*B*,*X*，用空格分开

---

#### 题解

**本题实质为有权有向图寻最短路问题**，每个关卡有两条路，一条连向下一关，一条连向指定关

故采用**dijkstra算法**进行求解*（本人第一次做题时脑抽用了dfs，超时惨烈）*

详解都写在了代码里 直接看吧111

如果对**邻接表**不熟悉的话简单标注一下各数组的意义

```c++
int h[N];  //用来指向该点所连的第一条边，N处填的是点（本题即为关卡），h[N]里面记录的是边
int e[N];  //该边指向的点
int ne[N];  //该边的下一条边
int w[N];  //该边的权重
int idx = 0;  //边下标
```

**ps**：因为本题的设定决定了不会出现重边和无法抵达终点的情况，

故省略了原本算法内重边的判断和无法抵达终点的判断

---

#### AC代码

```c++
#include<iostream>
#include<cstring>
#include<queue>

using namespace std;
typedef long long int LL;  //题目所给数据范围需要开longlong
const int N = 400020;  //因为每关都向外连两关，所以构建邻接表时开两倍最大值

int n;
int h[N],e[N],ne[N],w[N],idx = 0;  //构建邻接表
LL dist[N];  //记录每个点距离起点的路程

void add(int a, int b, int c) {  //头插法插入节点
    e[idx] = b;
    w[idx] = c;
    ne[idx] = h[a];
    h[a] = idx++;
}

LL dijkstra() {  //dijkstra堆优化求最短路算法

    //创建小根堆（优先队列）
    priority_queue< pair<int,int>, vector< pair<int,int> >, greater<> > heap;
    //将每点路程初始化为无穷远
    memset(dist, 0x3f, sizeof dist);
    //第一关路程为0
    dist[1] = 0;
    //第一个点入堆
    heap.push({0,1});  //因为小根堆默认以第一个数据排序所以数对格式为 {路程, 编号}
    
    while (!heap.empty()) {  //堆不空时循环
        int x = heap.top().second;  //记录当前路程最小的点
        heap.pop();  //移除该点
        for (int i = h[x]; i != -1; i = ne[i]) {  //遍历该点通向的所有点
            int j = e[i];  //记录通向的点的下标
            //比较“当前点到起点路程+当前点到通向点路程”和“通向点到起点路程”的大小
            if (dist[x] + w[i] < dist[j]) {
                //如果前者路程更小则更新通向点的最小路程并入堆
                dist[j] = dist[x] + w[i];
                heap.push({dist[j], j});
            }
        }
    }
    
    return dist[n];  //返回终点的路程
}

int main(){

    cin >> n;
    memset(h, -1, sizeof h);  //初始化邻接表
    
    for (int i = 1; i <= n-1; i++) {  //读入路线数据
        int a,b,c;
        cin >> a >> b >> c;
        add(i, i+1, a);
        add(i, c, b);
    }

    cout << dijkstra();  //输出答案

    return 0;
}
```
